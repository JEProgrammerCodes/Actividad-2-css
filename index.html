Identifica y describe las propiedades de los siguientes elementos CSS:

Tema 1-Texto
PROPIEDAD VALOR INICIAL DESCRIPCIÓN
color #f0f8ff La propiedad color en CSS se utiliza para definir el color del texto dentro de un elemento.
font-family Courier New', Courier, monospace a propiedad font-family define las fuentes que se aplicarán al texto. Si la primera fuente no está disponible, se utiliza la siguiente en la lista.
font-size medium (16px) Especifica el tamaño del texto. Puede definirse en píxeles, em, rem, puntos o porcentajes, ajustando la legibilidad y escala del contenido.
font-weight normal (400px) Define el grosor del texto. Puede tener valores como normal, bold, lighter, o números de 100 a 900, donde 400 es normal y 700 es negrita.
font-style italic Especifica el estilo de la fuente, permitiendo valores como normal (texto normal), italic (cursiva) y oblique (oblicua).
font-variant Courier New', Courier, monospace Controla el uso de variantes tipográficas, como small-caps, que transforma las letras minúsculas en mayúsculas pequeñas, proporcionando un efecto de estilo.
font 100 Es una propiedad abreviada que permite definir múltiples propiedades de fuente en una sola declaración, incluyendo font-style, font-variant, font-weight, font-size, line-height y font-family.
text-align center Especifica la alineación del texto dentro de un contenedor, permitiendo valores como left, right, center, y justify.
line-height normal Define la altura de línea del texto, que influye en el espacio entre líneas. Se puede especificar en unidades relativas (como em) o absolutas (como px), o como un número para multiplicar el tamaño de fuente.
text-decoration dashed Añade decoraciones al texto, como underline (subrayado), line-through (línea a través del texto) o overline (línea sobre el texto).
text-transform capitalize Controla la capitalización del texto, permitiendo valores como uppercase (convertir a mayúsculas), lowercase (convertir a minúsculas), capitalize (la primera letra de cada palabra en mayúscula), y none (sin transformación).
vertical-align auto Define la alineación vertical de un elemento en línea en relación a su línea base o a otros elementos en la misma línea. Los valores comunes incluyen top, middle, bottom, y baseline.
text-indent 0% Especifica la sangría de la primera línea de un párrafo. Se puede definir en unidades como píxeles (px), em (em), o porcentaje (%).
letter-spacing normal Define el espaciado entre caracteres en un texto. Se puede especificar en unidades de longitud, como píxeles (px) o em (em), permitiendo aumentar o disminuir el espacio entre letras.
word-spacing normal Especifica el espacio adicional entre palabras en un texto. Se puede definir en unidades de longitud como píxeles (px) o em (em), lo que permite aumentar o reducir el espaciado entre palabras.
white-space normal Controla cómo se manejan los espacios en blanco, saltos de línea y tabulaciones en un bloque de texto. Los valores incluyen: normal: colapsa espacios consecutivos. nowrap: impide los saltos de línea. pre: respeta espacios en blanco y saltos de línea. pre-wrap: respeta espacios en blanco pero permite saltos de línea. pre-line: colapsa espacios en blanco pero respeta saltos de línea.



Tema 2-Enlaces
CARACTERISTICAS PROPIEDADES DESCRIPCIÓN
Tamaño, Color y Decoración
Tamaño (font-size) Define el tamaño del texto. Ejemplo de valor: 16px, 1.5em, 100%.
Color (color) Especifica el color del texto. Ejemplo de valor: #ff0000, rgb(255, 0, 0), red.
Decoración (text-decoration) Añade decoraciones al texto, como subrayado o tachado. Ejemplo de valor: underline, line-through, none.



Decoración Personalizad
text-decoration-line Define la línea decorativa (subrayado, tachado, etc.).
text-decoration-color Establece el color de la decoración.
text-decoration-style Ajusta el estilo de la línea (sólida, ondulada, etc.).
text-decoration-thickness Controla el grosor de la línea.





Tema 3-Imagenes
Característica Propiedades Descripción
Anchura y Altura
Anchura (width) Define el ancho de un elemento.
Altura (height) Define la altura de un elemento.



Eliminar el Borde
border Elimina el borde de un elemento al establecerlo en none.





Tema 4-Listas
PROPIEDAD VALOR INICIAL DESCRIPCIÓN
list-style-type armenian Esta propiedad CSS define el tipo de marcador que se utiliza para las listas, como viñetas (disc, circle, square) o números (decimal, upper-alpha, lower-alpha), entre otros.
list-style-position inside Esta propiedad CSS determina la posición del marcador de la lista, ya sea dentro del contenido del elemento (inside) o fuera de él (outside).
list-style-image none Esta propiedad CSS permite especificar una imagen personalizada como marcador para los elementos de una lista en lugar de los marcadores predeterminados.
list-style armenian Es una propiedad abreviada (shorthand) en CSS que permite definir de forma conjunta el tipo de marcador, la posición y la imagen del marcador para una lista. Combina list-style-type, list-style-position y list-style-image en una sola línea.





Tema 5-Tablas
PROPIEDAD SE APLICA A VALOR INICIAL DESCRIPCIÓN
border-collapse Elementos de tablas (< table >). collapse Controla si los bordes de las celdas de la tabla están separados (separate) o colapsados en uno solo (collapse).
border-spacing Elementos de tablas (< table >). 0 cap Define el espacio entre los bordes de las celdas en una tabla cuando border-collapse está en separate.
empty-cells Celdas de tabla (< td >, < th >). hide Determina si se muestran los bordes y el fondo de las celdas vacías de la tabla.
caption-side Elementos de leyenda en tablas (< caption >). bottom Controla la posición de la leyenda de la tabla, permitiendo ubicarla en la parte superior (top) o inferior (bottom) de la tabla.
